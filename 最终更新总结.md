## 🚀 宝塔面板更新步骤（简化版）

### 第1步：备份（必须！）

在宝塔面板：
1. **数据库** → 找到 `fangame` → 点击 **备份**
2. **文件** → `/www/wwwroot/` → 压缩 `fangame-next` 文件夹

### 第2步：上传新文件

1. 在本地打包以下文件（**不要包含 node_modules**）：
   ```
   src/
   prisma/
   public/
   package.json
   package-lock.json
   next.config.ts
   tsconfig.json
   components.json
   .env
   其他配置文件
   ```

2. 压缩成 `update.zip`

3. 在宝塔面板：
   - 进入 **文件** → `/www/wwwroot/fangame-next/`
   - 上传 `update.zip`
   - 右键 → **解压** → **覆盖现有文件**

### 第3步：安装和构建

在宝塔面板终端执行：

```bash
cd /www/wwwroot/fangame-next

# 安装依赖
npm install

# 生成 Prisma 客户端
npx prisma generate

# 运行数据库迁移（应用新索引）
npx prisma migrate deploy

# 停止当前服务
pm2 stop fangame-next

# 清除缓存
rm -rf .next

# 重新构建
npm run build

# 启动服务
pm2 start fangame-next

# 查看状态
pm2 status
```

### 第4步：测试

访问网站，测试：
- ✅ 首页加载
- ✅ 用户登录/注册
- ✅ 游戏详情

## 🔍 故障排查

### 如果网站无法访问

```bash
# 1. 检查 PM2 状态
pm2 status

# 2. 查看日志
pm2 logs fangame-next --lines 50

# 3. 检查端口
lsof -i :3000

# 4. 重启服务
pm2 restart fangame-next
```

### 如果构建失败

```bash
# 清除缓存
rm -rf .next node_modules

# 重新安装
npm install

# 重新构建
npm run build
```

### 如果数据库连接失败

```bash
# 测试连接
PGPASSWORD='你的密码' psql -U fangame -h localhost fangame -c "SELECT 1;"

# 检查 PostgreSQL 状态
systemctl status postgresql
```
